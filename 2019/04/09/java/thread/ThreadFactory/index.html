ThreadFactory

线程的工厂类

源码分析

ThreadFactory就是一个接口，里面实现了获取线程的方法。在这个方法中，可以自定义线程的名字以及优先级和是否为守护线程等方法。

使用

ThreadFactory是工厂设计模式，是一种创建型设计模式，能够用于创建一个类或多个类所需要的对象。利用这个，可以集中的创建对象。集中创建对象带来的
好处有:

  • 能够很容易的改变类创建的对象或者创建对象的方式
  • 能够很容易的限制对象的创建
  • 能够很容易的生成有关对象创建的统计数据
